Detailed Description

Class that represents a Transformation Matrix, and stores the relevant data.

Constructor & Destructor Documentation

◆ Transform() [1/3]

Transform.Transform ( )
inline

Constructor that defaults the values, and has an identity matrix for the Matrix data.

The Scale will have both 'X' and 'Y' values set to 1.0. The Rotation will be 0.0. The Translation will have both 'X' and 'Y' values set to 0.0.

◆ Reset()

void Transform.Reset ( )
inline

Resets the values of this Transform to their defaults. The Matrix will also be set to its identity state.

The Scale will have both 'X' and 'Y' values set to 1.0. The Rotation will be 0.0. The Translation will have both 'X' and 'Y' values set to 0.0.
